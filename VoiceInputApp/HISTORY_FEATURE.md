# 对话记录永久保存功能

## 功能概述

为 Android 应用添加了完整的对话记录永久保存功能，所有发送的文本都会自动保存到本地存储，应用重启后仍然可以查看。

## 新增功能

### 1. 自动保存
- 每次发送文本时自动保存到本地存储
- 使用 SharedPreferences 持久化存储
- 最多保存 1000 条历史记录（可配置）

### 2. 历史记录管理
- **查看历史**: 应用启动时自动加载所有历史记录
- **搜索功能**: 点击顶部搜索图标，实时搜索历史记录
- **删除单条**: 点击历史记录右侧的删除按钮，或长按记录
- **清空所有**: 点击顶部删除图标，清空所有历史记录（需确认）

### 3. UI 增强
- 顶部工具栏新增搜索和删除按钮（仅在有历史记录时显示）
- 历史记录卡片右侧显示删除按钮
- 支持长按历史记录快速删除
- 所有删除操作都有确认对话框，防止误操作

## 技术实现

### 新增文件
1. `HistoryRepository.kt` - 历史记录数据仓库
   - 负责历史记录的增删改查
   - 使用协程进行异步操作
   - 支持搜索和分页查询

### 修改文件
1. `InputViewModel.kt` - 添加历史记录管理方法
   - `deleteHistoryItem()` - 删除单条记录
   - `clearAllHistory()` - 清空所有记录
   - `searchHistory()` - 搜索记录
   - `reloadHistory()` - 重新加载记录

2. `InputScreen.kt` - UI 界面更新
   - 添加搜索和清空按钮
   - 添加搜索对话框
   - 添加清空确认对话框

3. `HistoryItem.kt` - 历史记录组件增强
   - 添加删除按钮
   - 支持长按删除
   - 添加删除确认对话框

## 使用说明

### 查看历史记录
- 打开应用即可在主界面看到所有历史记录
- 历史记录按时间倒序排列（最新的在下方）

### 搜索历史记录
1. 点击顶部搜索图标
2. 在搜索框中输入关键词
3. 实时显示匹配的记录
4. 点击"关闭"恢复显示所有记录

### 删除单条记录
方法一：点击删除按钮
- 点击历史记录右侧的删除图标
- 在确认对话框中点击"删除"

方法二：长按删除
- 长按要删除的历史记录
- 在确认对话框中点击"删除"

### 清空所有记录
1. 点击顶部删除图标
2. 在确认对话框中点击"确定"
3. 所有历史记录将被永久删除

## 数据存储

- 存储位置: SharedPreferences (`voice_input_history`)
- 存储格式: JSON
- 最大容量: 1000 条记录
- 数据持久化: 应用卸载后数据会被清除

## 性能优化

- 使用协程进行异步 I/O 操作，不阻塞 UI 线程
- 历史记录超过 1000 条时自动删除最旧的记录
- 搜索功能使用内存过滤，响应迅速
