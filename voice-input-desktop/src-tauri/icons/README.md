# 图标文件说明

桌面端应用图标已准备就绪！

## 已生成的图标文件

- **icon.svg** - SVG 矢量图标（主图标）
- **icon.png** - 512x512 PNG 图标
- **icon.ico** - Windows ICO 格式图标（包含多种尺寸）
- **32x32.png** - 32x32 像素图标
- **128x128.png** - 128x128 像素图标
- **128x128@2x.png** - 256x256 高清图标
- **256x256.png** - 256x256 像素图标
- **512x512.png** - 512x512 像素图标

## 图标设计

图标采用现代设计风格：
- **主题**：麦克风（代表语音输入功能）
- **配色**：深紫色背景渐变 + 紫蓝色麦克风
- **风格**：简洁、专业、现代

## 使用方法

这些图标文件可以直接用于 Tauri 桌面应用编译。Tauri 会自动使用这些图标生成应用图标。

## 重新生成图标

如果需要修改图标样式，可以编辑 `generate_icons.py` 文件，然后运行：

```bash
python generate_icons.py
python generate_ico.py
```
